# `OpenHTTPA` Patent Policy

## Defensive Patent Strategy

The `OpenHTTPA` project includes several innovative technologies that are critical to the security of the confidential computing ecosystem:

1.  **AHL Semantic Binding**: Cryptographic binding of HTTP request semantics (Method, Path, Query) to the session MAC.
2.  **Heterogeneous TEE Synchronization**: Unified handshakes across multiple hardware providers (e.g., CPU + GPU).
3.  **Attested Agent Mesh (AAM)**: Transitive trust protocols for autonomous agent swarms.

These technologies are explicitly covered by the patent grant in **Section 3 of the Apache License 2.0**.

## Defensive Termination

This project leverages the Apache 2.0 "Patent Retaliation" clause.

> [!IMPORTANT]
> If any entity institutes patent litigation against the `OpenHTTPA` project or its contributors, their license to use these patents and this software is immediately terminated.

This ensures `OpenHTTPA` remains a protected "Safe Harbor" for all legitimate users and prevents patent trolls from weaponizing innovations against the open-source community.
